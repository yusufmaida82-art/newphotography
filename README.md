# Description
A personal photography website that showcases a collection of favorite 
photographs, a biography of the photographer, and a contact form for 
visitors to get in touch. The site is built with pure HTML and CSS, 
featuring a responsive layout using CSS Grid and Floats.

# Features
 Sticky navigation bar with smooth hover effects
 Hero section with a full background image
 Photo gallery built with CSS Grid (5 favorite photos)
 Biography section with float layout
 Contact form with focus and hover effects
 Fully responsive design for mobile and tablet
 Deployed and live on GitHub Pages
